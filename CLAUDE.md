# CLAUDE.md — правила работы с репозиторием

Этот файл читают AI-агенты и разработчики перед началом работы.

## Язык

Общение с AI — на русском. Документация — на русском.

## Ветки и пуши

- **В `main` напрямую не пушим никогда.** Все изменения — только через Pull Request.
- Репозиторий приватный, поэтому PR описывается текстом (заголовок + тело) без внешних ревьюеров.
- Каждый PR — одна логическая задача. Не мешать несвязанные изменения в один PR.

## Процесс валидации PR

- Перед созданием PR убедиться, что `pnpm lint` и `pnpm typecheck` проходят без ошибок.
- PR описывается понятно: что сделано и зачем.
- AI создаёт PR, но **не мержит его сам**.

## Мерж

**Мержит всегда владелец репозитория вручную.** AI не вызывает merge и не нажимает кнопку слияния ни при каких условиях.
