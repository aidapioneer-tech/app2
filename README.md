# aida-money-demo

Облачное приложение Битрикс24, встраивается во вкладку карточки сделки (`CRM_DEAL_DETAIL_TAB`)
и показывает финансовую картину сделки: план/факт, очищенный от НДС доход,
расходы по подрядчикам, маржу.

Источник данных — серверный REST-метод модуля `shef.reportbuilder` (репо
`aidapioneer-tech/aida`):

```
shef:reportbuilder.api.dealMoney.get { dealId }
```

## Установка

1. Зарегистрировать локальное приложение в Битрикс24:
   - URL: `https://<host>/`
   - URL установки: `https://<host>/install`
   - Скоупы: `crm`, `user_brief`
2. На странице установки (`/install`) приложение автоматически вызывает
   `placement.bind` для `CRM_DEAL_DETAIL_TAB` с обработчиком `/`.
3. После установки во вкладке «Деньги» карточки сделки видна
   финансовая сводка.

## Запуск разработки

```bash
pnpm install
pnpm dev      # http://localhost:3000
pnpm build
```

## Архитектура

- Nuxt 4 SPA на базе [`bitrix24/templates-dashboard`](https://github.com/bitrix24/templates-dashboard).
- Один REST-вызов на экран — вся агрегация на сервере.
- Категория сделки определяется по `deal.categoryId`:
  - `2` (клиент) → `Money/Client.vue`: шапка + таблица платежей клиента + список подрядчиков + итоги.
  - `3` (подрядчик) → `Money/Contractor.vue`: шапка + таблица платежей подряду + карточка родительской клиентской сделки + итоги.
- Учёт распределений (SP 1044) — отдельные колонки `Сумма`/`НДС` (факт)
  против `Всего`/`К оплате` (план); даты `Срок`/`Получено`.
